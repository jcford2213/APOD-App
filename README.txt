Astronomy Picture of the Day Date Range: 1995-06-16 - present


App Setup

  You need a NASA API key to use this site. 
    - Get a key at https://api.nasa.gov/
    - It is fast and free to do. Limit is 1,000 calls/hour

  In order for this app to work a .env file must be created with the following variables;

    - API_URL=https://api.nasa.gov/planetary/apod?api_key=YOUR_KEY

  Copy and Paste your api key over YOUR_KEY



Instructions to Run the Program
  - Backend
    - Open a terminal
    - Change directory to APOD-APP/backend
    - Run command: npx nodemon

  - Frontend
    - Open another terminal
    - Change directory to APOD-APP/frontend
    - Run command: npm run dev

  - Open a browser and type localhost:3000 in the address bar
